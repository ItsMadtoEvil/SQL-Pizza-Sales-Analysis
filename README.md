# 🍕 Pizza Sales Analysis using SQL Server

## Project Overview

This project analyzes pizza sales data using Microsoft SQL Server to generate business insights and KPIs.

The project demonstrates SQL skills including:

- Aggregate Functions
- GROUP BY
- ORDER BY
- CASE Statements
- DATE Functions
- Common Business KPIs
- Sales Analysis

---

## Dataset

The dataset contains:

- Order ID
- Pizza Name
- Pizza Category
- Pizza Size
- Quantity
- Order Date
- Order Time
- Unit Price
- Total Price

---

## Business Questions

✔ Total Revenue

✔ Average Order Value

✔ Total Orders

✔ Total Pizzas Sold

✔ Average Pizzas Per Order

✔ Daily Order Trend

✔ Hourly Order Trend

✔ Revenue by Pizza Category

✔ Revenue by Pizza Size

✔ Best Selling Pizzas

✔ Worst Selling Pizzas
